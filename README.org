** 遊び方：
- ~(load-file "tofu.el")~ で Emacs へ読み込み、
- ~M-x eval-buffer~ で読み込んだバッファを評価し、
- ~M-x tofu~ で実行します。

** UIの変更点：
- Emacs 24 以降では、画面の文字表示の大きさを選択できます。
- 晴れ、曇り、雨など、文字表記の一部をカラー化しています。
- コンピュータ側の動作を「ＣＰＵ」と表記しています。
